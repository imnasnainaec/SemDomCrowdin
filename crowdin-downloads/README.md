# Crowdin Downloads Folder

This folder contains files downloaded from Crowdin using the "Download" option.

## File Format

**Format:** XLF (XML Localization Interchange File Format)

**Extension:** `.xlf`

**Naming Convention:** `SemanticDomains.<language-code>.xlf` (with optional numbering or timestamp)

## Source

Files in this folder are obtained from Crowdin by:

1. Navigating to the translation project
2. Selecting the "Download" option
3. Choosing the desired language and file format

## Contents

The XLF files contain:

- Source text (original English Semantic Domain content)
- Target text (translated content)
- Translation metadata (IDs, attributes, structure)
- Semantic domain hierarchical organization
